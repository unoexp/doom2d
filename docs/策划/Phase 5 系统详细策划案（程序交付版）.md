# 一、战斗系统详细设计

## 1.1 战斗定位

《根与废土》的战斗**不是核心玩法**，而是生存探索的阻力层。
战斗的目的是让"回家"这件事有代价，让"准备"有意义。

**设计原则：**
- 战斗简洁直观，不追求格斗游戏深度
- 规避 > 正面对抗（鼓励玩家绕路而非硬刚）
- 夜间/深层敌人显著强于昼间/浅层
- 死亡惩罚明确（读档），增加紧张感


## 2.3 配方解锁逻辑

```
解锁方式有三种：
1. 默认解锁（UnlockedByDefault = true）
2. 图纸解锁（在地层中发现图纸 → 调用 CraftingSystem.UnlockRecipe()）
3. 进度解锁（建造特定模块后自动解锁一批配方）

进度解锁映射：
  制作台建成 → 解锁：合金镐、锈铁剑、兽皮套、睡袋、绳索
  冶炼炉建成 → 解锁：合金战锤、合金套（若有图纸）
  医疗站建成 → 解锁：解毒剂、兴奋剂
  根脉研究台建成 → 解锁：所有根脉系配方
```

# 三、角色成长系统

## 3.1 设计原则

采用**水平拓展 + 垂直精通**混合模式：
- 无传统等级系统，不升级不加点
- 通过**装备升级**（垂直）和**技能熟练度**（水平）实现成长
- 成长感来自庇护所升级、装备迭代、探索深度

## 3.2 技能熟练度系统

| 技能 | 提升方式 | 等级上限 | 每级效果 |
|------|---------|---------|---------|
| **挖掘** | 挖掘格子 | 10 | 挖掘速度+5%/级，体力消耗-3%/级 |
| **战斗** | 击败敌人 | 10 | 攻击力+3%/级，暴击率+1%/级 |
| **制作** | 成功制作 | 10 | 制作速度+8%/级，概率额外产出 |
| **烹饪** | 烹饪食物 | 10 | 食物恢复量+5%/级 |
| **采集** | 采集资源 | 10 | 采集量+8%/级 |
| **生存** | 存活时间 | 10 | 属性衰减-2%/级 |

### 熟练度经验公式

```
升级所需经验 = 基础经验 × (当前等级 ^ 1.5)
基础经验 = [PARAM: skill_base_exp] = 100

示例：
  Lv1→2: 100
  Lv2→3: 283
  Lv5→6: 1118
  Lv9→10: 2700

设计意图：前期快速提升手感，后期慢速提供长线目标
```

| 行为 | 基础经验 | 技能 |

## 6.1 设计原则

| 游戏时间 | 玩家应学会 | 引导方式 |
|---------|-----------|---------|
| 0-5分钟 | 移动、拾取 | 起始区域散落可拾取物品 |
| 5-15分钟 | 采集(砍树/挖矿) | 遮蔽所材料需求驱动 |
| 15-30分钟 | 建造 | 建造UI自动弹出需求清单 |
| 30-60分钟 | 生存管理 | 饥饿/口渴自然衰减 |
| 60-90分钟 | 战斗基础 | 夜间首次遇敌 |
| 90-120分钟 | 挖掘 | 储藏室建成解锁L1入口 |
