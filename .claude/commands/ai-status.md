# AI Company 状态面板

查看当前 AI 公司工作流的任务进度。

## 执行步骤

1. 用 `TaskList` 获取所有任务
2. 按状态分类统计（pending / in_progress / completed）
3. 输出状态面板：

```
═══════════════════════════════════════
  AI Company — 任务状态面板
═══════════════════════════════════════

  已完成：{N} / {总数}
  进行中：{N}
  待执行：{N}
  已阻塞：{N}

───────────────────────────────────────
  任务详情
───────────────────────────────────────
  [完成] #1 任务名称
  [进行] #2 任务名称
  [等待] #3 任务名称 (blocked by: #2)
  ...
═══════════════════════════════════════
```

4. 如果有进行中的任务，显示当前阶段（Phase 1-8）
5. 如果有阻塞的任务，分析阻塞原因并建议解决方案
